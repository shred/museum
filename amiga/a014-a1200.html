Commodore Amiga 1200

Configuration

  • Built: 1993 (second quarter)
  • Mainboard: Revision 1D.4
  • CPU: 68EC020 16MHz
  • OS: AmigaOS 3.2.2 (47.111)
  • Chipset: AGA
    • Alice: 8374 (date code 0793)
    • Lisa: 1204R0 (CBM, date code 1093)
    • Paula: 8364R7 (date code 0193)
    • Gayle: 391424-02 (rev 5, date code 0793)
    • Budgie: 391425-01 (rev 0, date code 0793)
  • RAM: 2 MB Chip RAM

Trivia

  • Came without floppy drive and without trapdoor cover.
  • Shield was there, but insulation sheet was missing. Previous owner used insulation tape instead, to avoid short circuits.

Known Issues

  • RCA jacks are oxidized and need replacement
  • One case screw is bent and needs replacement
  • Floppy drive button hole needs to be covered
  • Case illumination is missing

Restauration Works

  • Cleaning and whitening of the full case and keycaps.
  • Replaced case with A1200.net limited edition Arashi grey case.
  • Replaced original keycaps with white replica.
  • New LED board with custom LEDs (power: blue, floppy: green, harddisk: red).
  • Recapping. Replaced rusty crystal. Cleaned the board. Polished the modulator shield.
  • All screws were missing, including the D-Sub screws, and have been replaced.
  • Shield was there, but without insulator sheet. Replaced with simpler, but stronger replica IO shield.
  • Mouse port connector was broken, replaced by new one.
  • Removed E123C and E125C, to enhance compatibility with accelerator boards.
  • Keyboard wasn’t working properly. Replaced the keyboard connector, which fixed the problem. It was a fight to get the old connector out, but luckily the PCB stayed undamaged.
  • Added Indivision AGA MK3, GoEX drive (rev 2023) with display, SD to IDE card adapter.
