Sinclair ZX Spectrum

I bought this machine as a bundle of Sinclair hardware in February 2022. It came with the original box, PSU and cables. According to the price label on the box, the system was originally bought at Harrods for £129.95. It has been stored away for a long time, as it was quite filthy.

The computer was untested when sold. The diagnostics showed that the system was fully functional.

The PSU is an UK model that was later modified to get an Euro power plug. There is a label on it that says that the unit has been "re-checked after the recall of 26/2/83". Actually there has been a major recall of Sinclair PSUs after some faulty units with a poor galvanic isolation were found that were potentially hazardous.

After I only owned Issue Two boards so far, this is my first Speccy with an Issue Three board.

Configuration

  • Built: End of 1983
  • Mainboard: Issue Three
  • CPU: Z80A
  • ULA: 6C001E-6 (date code 8341, probably made in Phillipines)
  • RAM: 48KB
  • Came with an original box, cables and manuals.
  • Came with an original PSU with UK plug that was later changed to an EU plug (cleaned, untested, unrefurbished)

Restauration Works

  • Restauration was carried out between February and March 2022.
  • Thorough cleaning of the case. I also removed a security sticker that had been attached by the previous owner.
  • The keyboard membrane was intact, but one connector heavily bent, so I replaced it with a new membrane.
  • Faceplate replaced with a metallic blue one.
  • Replaced 7805 with Traco Power TSR-1 DC/DC converter, removed heatsink
  • Recapping, Composite Mod
  • Video calibration not possible/not required on Issue Three boards
