SpecNext Ltd. ZX Spectrum Next

Status: Working

This is not a real ZX Spectrum, but a modern FPGA implementation of the hardware, with many enhancements.

Configuration

  • Built: 2020
  • Mainboard: Kickstarter "Issue One", accelerated model (with RTC, WiFi, and Raspberry Pi Zero)
  • CPU: Z80N with up to 28MHz, implemented in FPGA
  • RAM: 2MB (1MB original + 1MB Renesas 71V424S10YG)
  • Sound: 3x AY-3-8912 (giving 9 synth channels), 2x PCM, Stereo
  • HDMI output

Trivia

  • This "Issue One" model is limited to 3010 units, so this is one of the most exclusive exhibits of my tiny home computer museum.

History

  • 2021-02-06: Added 1MB of RAM (Renesas 71V424S10YG)
  • 2020-04-23: Delivered!
