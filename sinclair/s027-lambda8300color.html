Lambda Electronics Lambda 8300 Colour Pack

Status: Not tested yet

This is a Colour Pack expansion for the Lambda 8300, a Sinclair ZX81 clone from Lambda Electronics Ltd of Hong Kong. It is connected to the expansion port and monitor output of the Lambda 8300, and provides its own TV and monitor output with a color signal. It has an internal color RAM that permits to set a foreground and background color for each 8x8 pixel character block, similar to the ZX Spectrum. The color palette consists of 8 colors.
