Lambda Electronics Lambda 8300 RAM

Status: Refurbished and working

This is a 16K memory expansion for the Lambda 8300. However, it can probably also be used on a ZX81.

Configuration

  • Built: 1984

Trivia

  • Came in a bundle with other ZX81 and ZX Spectrum extensions.
  • Uses 4116 type RAMs, but does not seem to have a DC/DC power converter for the required +12V and -5V lines. I guess the +12V is taken from the Lambda power, while the -5V is done by a DC/DC inverter. If that's true, this memory expansion won't work on a ZX81 though, because it only provides +9V.

Restauration Works

  • Whitened the case
  • Replaced all electrolytic capacitors. One original 1µF capacitor was replaced with a 2.2µF capacitor because I had no lower value on stock.
