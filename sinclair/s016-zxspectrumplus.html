Sinclair ZX Spectrum Plus

Configuration

  • Built: End of 1984
  • Mainboard: Issue 4B
  • CPU: Z80B
  • ULA: 6C001E-7 (date code 8445, probably made in Philippines)
  • RAM: 48KB (upper TMS4532-20NL3)
  • Composite mod

Trivia

  • Got this from a bundle of two ZX Spectrum+.
  • The previous owner added a power LED and began working on a monitor connector. He did it in a very professional and careful way. From the outside, these modifications look like they have always been there.
  • Uses a SN94459N as IC14.
  • Has a yellow power LED mod.

Known Issues

  • Membrane contact is broken and needs replacement.
  • Reset button and power LED needs to be resoldered.
  • Needs recapping.
  • Monitor modification is incomplete. The previous owner stopped modifying this machine at some stage, for unknown reasons.
