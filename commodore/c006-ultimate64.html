Gideons Logic Ultimate64

This is not a real C64, but a modern FPGA implementation of the hardware. It comes with an integrated tape and floppy emulator.

Configuration

  • C64-G case with white keyboard
  • Gideon's Logic Ultimate-64 Elite V1.4a, black PCB
  • FPGA implementation of original C64 hardware
  • SID: 6581 (date code 0184)
