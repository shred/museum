Commodore C64

Status: Refurbished and working

Configuration

  • Built: Q3 1984
  • Mainboard: 250407 Rev C
  • CPU: 6510 (date code 2284)
  • SID: 6581 (date code 3284)
  • VIC: 6569R3 (date code 3284)
  • RAM: 64KB
  • "Breadbin" case, original brown

Trivia

  • Came with an original manual.
  • All DIP chips are socketed in this machine.

Restauration Works

  • Cleaned the case, whitened it a bit in the sun.
  • Fully disassembled and cleaned the keyboard. The keycaps were whitened in the sun.
  • Recapped the board. Also replaced the 7805 with a Traco Power DC/DC converter.
  • CPU, SID and VIC got proper heatsinks.
  • Two case hinges were broken and replaced with 3D prints.
  • The original rubber feet got sticky after cleaning, and were replaced with new ones.
  • Full diagnostics test was successful. Keyboard test was successful.
