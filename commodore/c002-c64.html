Commodore C64

Status: Working, but needs overhaul

Configuration

  • Built: End of 1984 in England
  • Mainboard: Rev 250425
  • CPU: 6510 (date code 3684)
  • SID: 6581 (date code 3884)
  • VIC: (unknown, covered by heatsink)
  • RAM: 64KB
  • "Breadbin" case
  • Modification: Reset button, two Kernel ROM images (selectable by switch)

Trivia

  • Came with a 1541 floppy drive and an "elephant foot" PSU.
  • System was already partially recapped, and the 7812 regulator has a heatsink.
  • Has a reset button on the right side.
  • Has a ROM selector switch on the back. It switches between the standard C64 BASIC and "C64 Improved by EXOS V3" BASIC.
  • The floppy drive was modified by the previous owner. It has a blue LED and an USB connector.

Known Issues

  • Many screws were missing or replaced with screws of the wrong type. Some screw holes are broken out and need to be restaured (2x case, 3x PCB, 1x Keyboard).
  • The "Pound" key is not responding.
  • When running the diagnostics, the SID "noise" channels are inaudible.
  • Position of reset button might block some joystick connectors.

History

  • 2022-06-19: Stored for later refurbishment
  • 2022-04-04: Date of purchase
