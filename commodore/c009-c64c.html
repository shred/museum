Commodore C64C

Configuration

  • Built: 1988, week 46
  • Mainboard: Assy 250469 Rev 4 (short board with separate color RAM chip)
  • CPU: 8500 (date code 3488)
  • SID: 8580R5 (date code 3688)
  • VIC-II: 8565R2 (date code 3888, type misprinted as “8656R2”)
  • RAM: 64KB

Restauration Works

  • Mainboard was recapped and cleaned.
  • VIC and SID got heatsinks.

Trivia

  • I reassembled this C64-C from different components that have been purchased separately.
  • The case was originally planned for the Ultimate 64, but there was no way to properly mount the keyboard. The Ultimate 64 then moved to a classical “breadbin” case, and I had this C64 C case left.
  • The seller of the mainboard wished that the mainboard would get a new life. His wish was granted. ðŸ™‚
  • The VIC had a silver label “MOS 8565R2 3888 22” on it. When I removed it, I found “MOS 8656R2 3888” printed on the chip. This is a rare VIC with a misprint.

History

  • 2022-12-10:
    • Took the C64 C keyboard from the Ultimate 64.
    • Recapped the mainboard. Used a bipolar cap for the SID audio path.
    • Cleaned the mainboard.
    • VIC and SID got heatsinks.
    • Assembled the system using the mainboard, case, and keyboard.
    • Full harness diagnostics showed everything is working perfectly.
  • 2022-11-30: Bought a C64 C mainboard (250469, 252311 rev 4)
  • 2022-06-11: Bought a C64 C Case
