Commodore C64G

Configuration

  • Built: August 17th, 1988 in Western Germany
  • Mainboard: Assy 250469 Rev 4 (short board with separate color RAM chip)
  • CPU: 8500 (date code 2088)
  • SID: 8580R5 (date code 2288)
  • VIC-II: 8565R2 (date code 1988)
  • PLA: 251715-01
  • RAM: 64KB

Trivia

  • Came with an external "reset" expansion with machine-typed label.
  • Had two intact warranty seals when I got it.

Restauration Works

  • Case and keyboard cleaned. Case was a bit yellowed on a side, and whitened in the sun.
  • Recapping. (No voltage regulators on board for replacement.)
  • Heatsinks glued on CPU, VIC, and SID.
  • Digifix for 8580 SID, using a 820kΩ resistor for moderate digi sounds.
  • Cleaned up the PCB.
  • Machine thorougly tested by full diagnostics, no problems were found.
