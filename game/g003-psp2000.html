Sony PSP-2000

Status: Working

Configuration

  • Built: second half of 2007 (no date code found)
  • PlayStation Portable 2000 (original color: ice silver)
  • Motherboard: TA-085v1 (Slim 2000, Generation 2)
  • CPU: MIPS R4000, up to 333 MHz
  • RAM: 64 MB (main) + 4 MB (graphics)
  • Display: 480 x 272 Pixel, True Color
  • 32 GB MicroSD via MemoryStick Pro Duo adapter

Trivia

  • Was sold as working. After purchase, the seller remembered that the volume keys were broken.
  • This PSP seems to be a Japan import. It came with a Japanese manual, and probably has a Japanese regional code on the UMD. It's fine for me, as I will use the console for homebrew software only.

Restauration Works

  • Came with a cheap and fake "2400mAh" battery. Replaced it with a high quality replica battery.
  • Polished a lot of micro scratches from display. Unfortunately in vain because the original case could not be rescued.
  • Could not remove overturned screws, so I cut open the original case, and moved the interior to a transparent blue replica case. However, I kept the original joystick head, directional buttons, fire buttons, shoulder buttons, and memory stick cover.
  • Volume buttons were unresponsive, so I replaced the membrane of the control buttons.
  • Installed Infinity patch and LME CFW.
  • Applied non-glossy screen protector.
