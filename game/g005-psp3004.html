Sony PSP-3004

Configuration

  • Built: Fourth quarter of 2008 (date code 8D)
  • PlayStation Portable 3004, transparent black (original color: black)
  • Motherboard: TA-090v2 (Bright 3000, Generation 3)
  • CPU: MIPS R4000, up to 333 MHz
  • RAM: 64 MB (main) + 4 MB (graphics)
  • Display: 480 x 272 Pixel, True Color
  • 32 GB MicroSD via MemoryStick Pro Duo adapter

Trivia

  • Was sold in a bundle of two broken PSPs. My first intention was to keep them as spare part donors, but then I felt pity for the poor machines.

Restauration Works

  • Moved everything into a new, clear black replica case (which actually looks rather brownish). I reused the white direction and control keys from the case of another donor (not part of the museum).
  • WiFi switch was broken off, and the device was unable to read from Memory Stick. I replaced the mainboard with one from the same donor.
  • Display was broken, so I used the one from the other device.
  • Installed latest firmware, Infinity patch and LME CFW.
  • Applied non-glossy screen protector.

History

  • 2024-03-20
    • UMD drive tested and confirmed working
  • 2023-11-28:
    • Replaced mainboard with a donor’s PSP-3000 board
    • Updated to firmware 6.61 and installed CFW
  • 2022-11-26:
    • Reassembled the system into a clear black replica case. Unfortunately the replica case already came with a lot of tiny scratches on the backside.
    • Moved the LCD of the other PSP-3000 into this device.
  • 2022-11-13: Disassembled
    • Original case was missing the back frame with WiFi switch, and the battery cover.
  • 2022-10-09: Date of purchase
